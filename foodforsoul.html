Food for Soul

The project is to analyse, define and develop a project strategy for the selected brand. In this case the brand to work on is Food for Soul. The strategy will show how the brand could leverage digital media, design and communication to achieve brand goals and engage target users.

The output of this project requires research, develop and present a highly focused digital proposal.
The project must include:
Research & benchmarking
Strategy for digital communication
Technical strategy for development
UX & UI proposal
Working prototype : Mid/High fidelity prototype using Html, CSS, Js, etc.
