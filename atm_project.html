ATM Application Restyling

The project is the restyling of an ATM application. The main part of the project consisted of UX, UI and prototype building of the application.
UX : To analyse the flow and application structure on the key functions, and it has to be responsive.
UI : To study how to improve the look and feel of the WebApp. Considering the text styles, icons, colour and other relevant design elements.
Prototype : To build the full design for mobile application prototype.

The project is a Group project of four groupmates. Our goal is to restyle the existing application for users to use the application more efficiently. Before the designing of the application, we did a user testing on ATM kiosk to observe users behaviour while using the kiosk.

The user testing assisted us to know what is the priority for users when they are using the ATM kiosk, so that we could apply the important and necessary parts into the building of ATM application prototype. We started from the discussion of the existing application and how could we improve. We share our ideas on the design layout, functionally font styles, icons and colour. We divided the work on research part and designing the prototype.
My Role : UX and UI designer , Prototype builder

Before we create the protoype of ATM APP, we did a user testing on ATM kiosk, to test users behaviour. In order to help us develop the app.

Design Features of the App

Here is the style sheet for typography and colour we decided to use in the app. Also the main menu features of the app.

The main menu provides an easy access to the most substantial features. The gradient background is our new approach to create a distinct colour identity for the app.

Personal Profile Users can create their personal profile where they can have their purchase details and history recorded for a faster payment subsequently.

Customization Users also have the ability to customize their frequently purchased tickets by editing from the default. After payment of the tickets, there is an option for users to sign up in order to save their purchase details for quicker purchase in the future.

Prepurchase and Repurchase Signed up users would be able to find a record of their travel card where they could check the availability date and to pre-purchase for next month. All users could repurchase their tickets from their purchase history which is in my trips feature.

QR Code After purchasing users will receive a QR code which would go to their purchase baskets, where they could find their QR code again by going into mytrip menu. Qr code will activate when the user start to use, The app will time the journey, and the ticket is valid for 90 minutes after activated.
